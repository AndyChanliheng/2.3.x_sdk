## 3. 网络状态对应指示灯

+ 配网中:ez配网，指示灯快闪(250ms闪烁)和AP配网，指示灯慢闪(1500ms闪烁)

+ 其他wifi状态：wifi指示灯显示继电器状态；


## 4. 其他功能

+ wifi模式为GWCM_LOW_POWER，长按按键5秒进入快闪模式；
+ 产测方案，未授权退出产测。以授权成功，进入产测wifi指示灯慢闪，按一下按键继电器动作动作3次，每次间隔500ms.
